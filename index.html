Sync Commands

● Ready

Run this command locally in your repository to create the commit and generate your green dot:

python sync_challenge.py --title="..." --platform="..."

How to set this up:

  1. Initialize a new, empty repository on GitHub named daily-coding-challenges.
  2. Clone it to your local machine and copy the sync_challenge.py file into its root directory.
  3. Run the copied command in your terminal inside the repository.
  4. Push changes (git push) to update your GitHub graph instantly.
